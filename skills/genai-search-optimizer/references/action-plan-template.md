# Action Plan Template

Use this template to deliver the audit result as a prioritized, citation-backed plan. The goal is a plan the user can execute, not a list of observations.

## Header

```text
Page or site:
Audited URL or content target:
Audit date:
Auditor / agent:
Goal stated by user:
Baseline state (one paragraph):
```

## Blockers

Items that must be fixed before any content work can move the needle. Usually baseline eligibility, crawl, or indexing problems.

```text
- Finding:
  Severity: blocker
  Rationale:
  Citation:
  Next action:
  Effort estimate:
  Owner:
```

Repeat the block for each blocker.

## High-impact actions

Items likely to move visibility once blockers are clear. Usually content originality, primary technical structure, and primary media.

```text
- Finding:
  Severity: major
  Rationale:
  Citation:
  Next action:
  Effort estimate:
  Owner:
```

## Polish actions

Small improvements that help but are unlikely to move visibility on their own.

```text
- Finding:
  Severity: minor or polish
  Rationale:
  Citation:
  Next action:
  Effort estimate:
  Owner:
```

## Myth-bust list

Tactics the user should stop or avoid, with citations.

```text
- Tactic to stop:
  Why it does not work:
  Citation:
  Foundational SEO equivalent to do instead:
```

## 30 / 60 / 90 day staging (optional)

Use when the user wants a roadmap.

```text
Days 0-30:
- Fix blockers.
- Ship one high-impact content rewrite as a proof of concept.
- Set up Search Console monitoring for the audited URL.

Days 31-60:
- Ship remaining high-impact actions.
- Start polish actions in batches.
- Compare Search Console performance to baseline.

Days 61-90:
- Audit a second page or content type using the same workflow.
- Decide whether the rewrite patterns should become editorial policy.
- Reconfirm baseline eligibility for the broader site.
```

## Open questions

```text
- Missing data needed to finish the audit:
- Decisions the user must make before next steps can be planned:
- Items the agent could not verify without browser access, Search Console access, or analytics:
```

## Closing note for the user

```text
This audit is based on Google Search Central's public guidance on generative AI search, helpful content, and technical SEO. It is unofficial and does not guarantee outcomes in AI Overviews, AI Mode, or classic Search. Visibility depends on many factors outside any single page.

Citations: list the specific Search Central pages referenced in this audit.
```
